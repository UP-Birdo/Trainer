Gut zu wissen

App-Updates löschen keine Daten. Der Code kommt von der Website, deine Daten liegen auf diesem Gerät.
Neues Handy oder Safari-Daten gelöscht? Auf dem Anmeldebildschirm „Sicherung wiederherstellen“ und die Datei trainer-sicherung-….json wählen.
Passwort vergessen? Beim Anmelden „Passwort vergessen?“ und den notierten Wiederherstellungscode eingeben.
Ohne Passwort UND ohne Code sind die Daten endgültig unlesbar — das ist der Preis der Verschlüsselung.
